# Design Guidelines Reference Lookup

Use this guide to find the right reference files for any design review topic.
All files are in `references/hig/` relative to this skill.

These references contain universal design principles adapted from Apple's HIG.
They apply to any mobile or desktop framework (Flutter, Tauri, Electron, React Native, etc.).
